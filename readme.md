## Proyecto Reduxstagram

Publicaciones con foto estilo Instagram

#### Tecnología usada:
```
React + Redux + webpack
```
#### Instalar dependencias:
```
yarn install
```

#### Arranque
Ejecuta `yarn start` y abre <localhost:7770> en el navegador
